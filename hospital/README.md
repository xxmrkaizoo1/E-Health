# Hospital Web Application

## Overview
The Hospital Web Application is designed to provide a user-friendly interface for students and organized staff to access health-related information and services. The application features an auto-slideshow for advertisements, enhancing user engagement and promoting health services.

## Project Structure
```
hospital
├── index.php          # Main entry point of the web application
├── ads                # Directory containing advertisement files
│   ├── ad1.php       # Content for the first advertisement
│   ├── ad2.php       # Content for the second advertisement
│   ├── ad3.php       # Content for the third advertisement
│   └── ad4.php       # Content for the fourth advertisement
├── img                # Directory containing advertisement images
│   ├── ad1.jpg       # Image for the first advertisement
│   ├── ad2.jpg       # Image for the second advertisement
│   ├── ad3.jpg       # Image for the third advertisement
│   └── ad4.jpg       # Image for the fourth advertisement
├── css                # Directory for CSS styles
│   └── style.css      # Styles for the web application
├── js                 # Directory for JavaScript files
│   └── ads-slideshow.js # JavaScript for the advertisement slideshow
└── README.md          # Documentation for the project
```

## Features
- **Auto-Slideshow for Advertisements**: The application includes an auto-slideshow feature that cycles through four advertisements, enhancing visibility and engagement.
- **Responsive Design**: The web application is designed to be responsive, ensuring a seamless experience across various devices.
- **User-Friendly Interface**: The application provides an intuitive interface for users to navigate and access information easily.

## Setup Instructions
1. **Clone the Repository**: Clone the project repository to your local machine.
2. **Install Dependencies**: Ensure you have a local server environment set up (e.g., XAMPP, WAMP) to run PHP applications.
3. **Place Files**: Place the project files in the appropriate directory of your local server.
4. **Access the Application**: Open your web browser and navigate to `http://localhost/hospital/index.php` to view the application.

## Usage
- Users can select their login type (Organized or Student) to access relevant services.
- The advertisement section will automatically cycle through the ads, providing information about health services and promotions.
